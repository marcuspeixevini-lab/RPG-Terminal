O RPG Terminal é um projeto desenvolvido em Swift cujo objetivo é simular um jogo de RPG totalmente executado no terminal. A proposta é oferecer uma experiência simples, porém funcional, em que o usuário pode interagir com diferentes sistemas clássicos de jogos de role-playing, ao mesmo tempo em que pratica conceitos fundamentais da linguagem Swift. O projeto utiliza funções, estruturas condicionais, loops, manipulação de variáveis e entrada de dados pelo console, tornando-se uma base eficiente tanto para aprendizado quanto para futuras expansões do sistema.

Durante a execução, o jogador acessa um menu principal que permite navegar por diversas funcionalidades. A loja oferece itens que podem ser adquiridos usando o ouro acumulado. O sistema de batalhas simula confrontos diretos com monstros, nos quais o jogador pode ganhar ouro e nível ou perder vida dependendo do desempenho. O inventário reúne todos os itens obtidos, além de exibir o estado atual do personagem. A exploração adiciona elementos de aleatoriedade ao jogo, podendo resultar em descobertas de recursos, itens raros ou pequenos encontros que afetam a vida do personagem. O sistema de missões avalia o progresso do jogador em determinados objetivos e concede recompensas quando eles são concluídos. Há também a opção de treinamento, que permite evoluir o nível do personagem mediante o gasto de uma quantidade específica de ouro.

O RPG Terminal pode ser executado tanto no Xcode quanto diretamente pelo terminal do macOS. No Xcode, basta criar um projeto do tipo “Command Line Tool”, selecionar Swift como linguagem e inserir o código no arquivo principal antes de compilar e executar. Já no terminal, o usuário pode salvar o código em um arquivo Swift e executá-lo utilizando o comando apropriado do compilador Swift. Em ambos os casos, a interação ocorre por meio das opções digitadas pelo usuário no console, garantindo uma experiência completamente textual.

O projeto foi desenvolvido com uma estrutura clara e objetiva, permitindo fácil entendimento da lógica geral e oferecendo um ponto de partida sólido para quem deseja aprender Swift ou construir sistemas de jogo mais complexos. Além disso, sua organização permite futuras expansões, como inclusão de novos itens, eventos, mecânicas ou melhorias no sistema atual, mantendo a flexibilidade necessária para evolução contínua.
